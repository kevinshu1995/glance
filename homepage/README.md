# Homepage Configuration

Personal services dashboard and bookmarks manager.

## Configuration Files

- `services.yaml` - Define self-hosted services
- `bookmarks.yaml` - Browser bookmarks
- `settings.yaml` - Application settings
- `docker.yaml` - Docker integration
- `widgets.yaml` - Widget configuration

## Features

- Service status monitoring
- Quick bookmarks
- Docker integration
- Real-time monitoring
